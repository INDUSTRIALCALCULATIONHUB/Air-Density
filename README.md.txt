# Air Density Calculator for Fan Design

This Python program calculates atmospheric pressure and air density based on:

- Altitude (m above mean sea level)
- Gas temperature (°C)

It is useful for:
- ID / FD fan design
- HVAC calculations
- General engineering applications

---

## 📌 Method Used

1. Atmospheric pressure is calculated using the ISA (International Standard Atmosphere) model.
2. Air density is calculated using the ideal gas law:

    rho = P / (R * T)

---

## ⚙️ Inputs

- Altitude (meters)
- Temperature (°C)

---

## 📊 Outputs

- Atmospheric Pressure (Pa)
- Air Density (kg/m³)
